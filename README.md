# MISHOS-Rh1

Dashboard วิเคราะห์ข้อมูลผลงานบริการ/การเงินจากระบบ MISHOS (สปสช.) เขตสุขภาพที่ 1
ครอบคลุมปีงบประมาณ 2560-2569 · 131 หน่วยบริการ · 8 จังหวัด

4 หน้า:
- `index.html` — เทรนด์ภาพรวมทั้งเขต/รายจังหวัด/รายโรงพยาบาล
- `hospital.html` — วิเคราะห์เชิงลึกรายโรงพยาบาล เทียบกลุ่มอ้างอิง (GroupName/ขนาด/จังหวัด/เขต/เลือกเอง)
- `rank.html` — จัดอันดับเทียบกลุ่ม
- `risk.html` — เงินเสี่ยง (AdjRW ส่งเกิน 91 วัน, ความผิดพลาดข้อมูลเบิก)

ข้อควรทราบเมื่อตีความตัวเลข: เดือน (`month_ym`) ใช้ปี พ.ศ., เดือน ก.ย. ทุกปีมียอดเคลมส่งช้าปนอยู่สูงผิดปกติ,
รายงานผู้ป่วยใน (IP) ปีงบ 2560-2562 เป็นช่วงเริ่มระบบไม่ใช้เทียบอัตราโต, ปีงบ 2569 มีข้อมูล ~9 เดือน
